**功能**:

- ✅ 测试步骤记录
- ✅ 操作记录
- ✅ 断言结果记录
- ✅ 失败自动截图
- ✅ Allure 报告集成

**使用示例**:
```python
from infra.logging.structured_logger import get_logger
from infra.reporting.screenshot_on_failure import take_screenshot

# 结构化日志
logger = get_logger("TestExecution")
logger.info("测试开始", test="login_test")
logger.step("输入用户名", status="start")
logger.action("type", "username_input", "admin")
logger.assertion("用户名正确", True)

# 失败自动截图 (pytest hook 自动触发)
# 或手动截图
take_screenshot("before_submit", "提交前的页面状态")
```
